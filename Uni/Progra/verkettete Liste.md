Jede Liste beginnt mit einer head Node.
Eine Node beinhaltet immer einen Wert beispielsweise eine Zahl und zeigt auf die nachfolgende Node, nur die letzte Node der Liste zeigt auf keine nachfolgende Node.

verkettete Listen sind in der Regel nicht sortiert.
