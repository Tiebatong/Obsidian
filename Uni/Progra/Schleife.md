
## for Schleife

Der Code im Schleifen body wird so oft ausgeführt, bis die Bedingung im Schleifen head nicht mehr erfüllt ist.
Geeignet wenn man etwas eine bestimmte Anzahl oft ausführen möchte, oder wenn man selber bestimmen möchte um wie viel inkrementiert oder dekrementiert wird.

## for each Schleife

Durchläuft jedes Element eines Arrays genau einmal.
Geeignet um ein Array auszugeben, oder über ein Array zu iterieren, aber nicht um Arrays zu manipulieren.

for (Datentyp name: Arrayname) {
	Code
}

![[Pasted image 20260211175419.png]]


## while Schleife
Wird so lange ausgeführt, bis die Bedingung im Schleifen head false ist.
Führt schnell zu endlos Schleifen.
Geeignet wenn vorher nicht absehbar ist, wie oft die Schleife ausgeführt werden muss.
