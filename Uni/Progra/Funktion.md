
eine Funktion ist ein Code Abschnitt, der mehrfach von verschiedenen Stellen im Programm aufgerufen werden kann.

Die meisten Funktionen bekommen [[Argument]] übergeben.

Eine Funktion hat immer einen [[Rückgabewert]], also int, float, String oder, wenn die Funktion nichts zurückgeben soll void.

Eine Funktion sollte nur eine einzige Sache tun.
