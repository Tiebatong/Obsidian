
Binary search kann nur auf bereits sortierte Arrays angewendet werden.

Binary search zerlegt ein [[Array]] in zwei Hälften, prüft dann ob die gesuchte Zahl größer oder kleiner ist.

Ist die gesuchte Zahl größer als die Zahl in der Mitte wird als nächstes die obere Hälfte untersucht, ist die gesuchte Zahl kleiner wird die untere Hälfte untersucht.
